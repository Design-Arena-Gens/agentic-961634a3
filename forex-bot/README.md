# Gemini Forex Autobot

An autonomous MetaTrader 5 execution engine orchestrated via Next.js and powered by Gemini AI for market analysis.

## Environment

Create an `.env.local` file with the following variables:

```
GEMINI_API_KEY=
METAAPI_TOKEN=
METAAPI_ACCOUNT_ID=
```

## Scripts

- `npm run dev`
- `npm run build`
- `npm run start`
- `npm run lint`
